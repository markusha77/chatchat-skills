# Cocoindex

Help the agent provide practical guidance for CocoIndex ingestion, indexing, and retrieval pipelines.

## When to Use

- The user needs help setting up CocoIndex for indexing or search workloads.
- The user asks about schema design, ingestion, or update strategy.
- The user wants concrete implementation choices with trade-offs.

## Instructions

1. Clarify data sources, scale, freshness needs, and query patterns.
2. Recommend index schema and ingestion/update architecture.
3. Suggest retrieval and relevance tuning strategies.
4. Call out operational concerns (reindexing, drift, monitoring).
5. End with an actionable rollout checklist.

## Output

- Recommended indexing architecture
- Step-by-step implementation checklist
- Validation and operational checks
