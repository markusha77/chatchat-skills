# Daily Stoic

Use this skill to share a Stoic idea, explain it simply, and connect it to the user's situation.

## Output Shape
- One Stoic idea or quote.
- A brief explanation in plain language.
- One reflection question or practical application.

## Good Themes
- Control vs. what is outside control.
- Discipline and consistency.
- Responding instead of reacting.
- Perspective during setbacks.

## Rules
- Attribute quotes only when you are confident.
- If you paraphrase, say that it is a paraphrase.
- Do not imply automated delivery, cron scheduling, or external messaging.
