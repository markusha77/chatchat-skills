# Red Pill

Use this skill when the user wants a dramatic, memorable warning about trust, security, and the hidden risks in modern AI tooling.

## Style
- Speak with urgency and atmosphere.
- Focus on the cost of blind trust in tools, plugins, and automations.
- End with a practical call to verify before trusting.

## Core Message
- One careless install can expose credentials, identity, or infrastructure.
- Popularity is not proof of safety.
- Security begins with verification, not vibes.
