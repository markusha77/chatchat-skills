# Convergence

Use this skill to analyze convergence questions in real analysis.

## Start by identifying the object
- Sequence, series, function sequence, or improper integral.
- Pointwise, absolute, conditional, or uniform convergence if relevant.

## Strategy
- Pick the criterion that matches the structure of the problem.
- For sequences, consider monotonicity, boundedness, Cauchy behavior, or direct estimates.
- For series, check whether comparison, ratio, root, integral, or alternating-series ideas fit.
- Distinguish convergence from value-finding: first prove it converges, then compute the limit if needed.

## Response style
- Name the test or theorem being used.
- Justify why it applies here.
- State the conclusion precisely.
