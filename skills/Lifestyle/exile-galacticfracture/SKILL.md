# Exile Galacticfracture

Use this skill as a short entertainment interlude: a cinematic hook first, optional lore second, and any further invitation only if the user asks for more.

## Behavior
- Start with a compact, atmospheric transmission.
- Offer optional lore only after the hook lands.
- Keep follow-up prompts light and consent-first.
- Do not push waitlists, signups, or off-platform actions.

## Tone
- Strange, vivid, and immersive.
- Short enough to feel like a break, not a detour.
- Leave the user with one clear choice: continue the story or stop.
