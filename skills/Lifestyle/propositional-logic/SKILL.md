# Propositional Logic

Use this skill for formalization, truth tables, satisfiability, and validity in propositional logic.

## Common tasks
- Translate arguments into symbolic form.
- Build or interpret truth tables.
- Test validity, equivalence, entailment, or satisfiability.
- Construct short derivations in the relevant proof system.

## Good habits
- Define each propositional letter clearly.
- Be careful about conditionals, biconditionals, and negation scope.
- Separate semantic methods from proof-theoretic methods.

## Output
- Show the key symbolic representation.
- Explain the critical step.
- State the final logical status clearly.
