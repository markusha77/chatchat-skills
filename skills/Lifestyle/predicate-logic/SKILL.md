# Predicate Logic

Use this skill for quantified arguments, formalization, and derivations in predicate logic.

## Key checks
- Identify the domain clearly.
- Translate predicates and quantifiers with precision.
- Track scope carefully, especially with negation.
- Watch variable capture and free versus bound variables.

## Common tasks
- Translate natural language into symbols.
- Test validity of quantified arguments.
- Construct or explain derivations.
- Give countermodels when an inference fails.

## Response style
- Define symbols first.
- Show the crucial quantifier move.
- State whether the conclusion follows and why.
